# Torneo Fácil

Aplicación web para organizar campeonatos deportivos en centros educativos. Permite crear equipos, generar fixtures, registrar resultados y gestionar fases en tiempo real, desde cualquier celular y sin necesidad de instalación.

🔗 **[Acceder a la app](https://diegogonzalezuy.github.io/campeonatos)**

---

## Funcionalidades

- Tres formatos de torneo: todos contra todos, dos fases + final, y eliminación directa
- Generación automática del fixture según la cantidad de equipos
- Carga de resultados partido a partido
- Tabla de posiciones en tiempo real
- Generación de informe exportable (copiable para WhatsApp u otras apps)
- Guardado automático en el navegador: si cerrás la app o apagás el celular, el campeonato se recupera al volver a abrirla
- Validaciones de datos: detecta letras en campos numéricos, nombres duplicados y valores fuera de rango

## Tecnologías

- HTML, CSS y JavaScript
- Sin dependencias de servidor ni base de datos
- Desplegada en GitHub Pages
- Persistencia local mediante localStorage

## Estado del proyecto

En uso activo y mejora continua a partir del feedback de docentes.
